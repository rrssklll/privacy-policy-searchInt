Политика конфиденциальности
для Telegram-бота SearchInt

(соответствует 152-ФЗ «О персональных данных» и иным нормам Российской Федерации)

1. Общие положения

1.1. Бот предоставляет услуги поиска информации в открытых источниках по следующим критериям:

  • Поиск по имени
  • Поиск авто по Российской Федерации (гос.номер, VIN)
  • Поиск в социальных сетях: Instagram, ВКонтакте, Facebook, TikTok, Одноклассники
  • Поиск по номеру телефона, email, Telegram-аккаунту
  • Поиск по кадастровому номеру
  • Поиск по тегам в телефонной книжке
  • Поиск по юр.лицам и ИНН
  • Поиск данных водительского удостоверения
  • Поиск фото человека, фото автомобильного номера
  • Анализ стикеров/эмодзи

1.2. Оператор: rrssklll

2. Какие данные обрабатываются

2.1. Данные пользователя:

  • User ID, логин Telegram, текст запросов (включая ФИО, номера телефонов, email, VIN, кадастровые номера, ИНН, данные водительского удостоверения)

2.2. Биометрические данные:

  • Фото человека

2.3. Данные из открытых источников:

  • Публичная информация из соц.сетей, государственных реестров, СМИ (п. 10 ст. 6 152-ФЗ)

2.4. Технические данные:

  • IP-адрес, метаданные (в обезличенной форме)

3. Цели обработки

3.1. Обработка данных осуществляется для:

  • Поиска информации по указанным критериям
  • Предоставления результатов на основе открытых источников
  • Технической поддержки работы Бота

4. Правовые основания

4.1. Обработка персональных данных пользователя:

  • На основании согласия (ст. 9 152-ФЗ) и договора на услуги (ст. 6 152-ФЗ)

4.2. Обработка данных из открытых источников:

  • п. 10 ст. 6 152-ФЗ (общедоступные данные)

4.3. Биометрические данные (фото):

  • Только с письменного согласия субъекта (ст. 11 152-ФЗ)

5. Сроки хранения

5.1. Данные пользователя (логин, запросы):

  • До отзыва согласия

5.2. Биометрические данные:

  • Немедленное удаление после обработки, если иное не согласовано

5.3. Данные из открытых источников:

  • Не сохраняются, кроме случаев, предусмотренных законом

6. Меры защиты

6.1. Оператор применяет:

  • Шифрование передаваемых данных
  • Ограничение доступа к биометрическим данным
  • Регулярные проверки системы безопасности (ст. 19 152-ФЗ)

7. Права пользователей

7.1. Пользователь вправе:

  • Отозвать согласие на обработку персональных и биометрических данных
  • Требовать удаления фото, номеров телефонов, иных данных (ст. 14, 21 152-ФЗ)
  • Направлять запросы на Telegram-аккаунт @rrssklll

8. Особые условия

8.1. Поиск по юр.лицам и ИНН:

  • Данные юр.лиц обрабатываются в рамках закона о коммерческой тайне (ст. 3 98-ФЗ)

8.2. Поиск по соц.сетям:

  • Бот использует только публично доступные данные
  • Пользователь обязуется не нарушать права третьих лиц

8.3. Фото человека/авто:

  • Результаты поиска фото предоставляются только из открытых источников
  • Хранение фото запрещено

9. Ответственность

9.1. Пользователь несет ответственность за:

  • Законность своих запросов и использование полученных данных (ст. 13.11 КоАП РФ, ст. 137 УК РФ)
  • Нарушение прав третьих лиц включая право на неприкосновенность частной жизни (ст. 24 Конституции РФ)

9.2. Оператор не несет ответственности:

  • За содержание, точность и актуальность информации из открытых источников
  • За незаконное использование данных пользователями
  • За ущерб от ошибок или устаревших сведений

9.3. Условия предоставления данных:

  • Данные предоставляются «как есть» (ст. 434 ГК РФ)
  • Оператор не гарантирует их соответствие целям пользователя

10. Отказ от гарантий

10.1. Бот осуществляет поиск исключительно в открытых источниках:

  • Оператор не проверяет законность публикации информации
  • Не влияет на содержание сторонних ресурсов
  • Не отвечает за действия пользователей, нарушающие законодательство РФ

11. Заключительные положения

11.1. Изменения в Политику:

  • Вступают в силу с момента публикации в Боте

11.2. Продолжение использования:

  • Считается согласием с актуальной версией Политики

11.3. Контакты:

  • Telegram-аккаунт @rrssklll
